Dracula

A dark theme for Textmate and 51+ apps

TextMate

Install using Git

If you are a git user, you can install the theme and keep up to date by cloning the repo:

$ git clone https://github.com/dracula/textmate.git

The default branch is for TextMate 2. If you need to use the TextMate 1 version, you need to checkout the v1 branch.

$ git checkout v1

Install manually

v2: Download using the GitHub .zip download option and unzip them.

v1: Download using the GitHub .zip download option and unzip them.

Activating theme

Just open the Dracula.tmTheme file using TextMate app.
