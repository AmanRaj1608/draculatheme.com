Dracula

A dark theme for CotEditor and 80+ apps

CotEditor

Install

Download using the GitHub .zip download option

Activating theme

  1. Start CotEditor
  2. Go to Preferences -> Appearance
  3. Click the gear icon and then select Import...
  4. Find and select the Dracula.cottheme file and click Import
  5. Click the Dracula theme to apply it
  6. Embrace the darkness
