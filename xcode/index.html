Dracula

A dark theme for Xcode and 89+ apps

Xcode

Install manually

  1. Download using the GitHub .zip download option and unzip them.

  2. Create the custom themes folder:

    $ mkdir ~/Library/Developer/Xcode/UserData/FontAndColorThemes/

  3. Move Dracula.xccolortheme file to this custom themes folder.

Activating theme

  1. Xcode > Preferences > Fonts & Colors
  2. Select the Dracula theme
